## What it is
Figure it Out turns Claude Code into a supreme problem-solving engine. It classifies any blocker, selects an intelligence mode, deploys a 21-agent strike force, and delivers a battle-tested solution that survives adversarial inputs and cost constraints.

## How it behaves (quickstart)
1. **Hear the trigger** (explicit or any detected blocker).
2. **Classify the problem DNA** (domain, type, constraint, complexity, urgency).
3. **Pick intelligence mode(s)** and escalation level.
4. **Deploy agents** (research + engineering + quality) in parallel.
5. **Verify** with TESTER + ADVERSARY + GUARDIAN.
6. **Deliver**: working artifact/commands, how to run, proof, recovery/rollback, predictive FAQ.
7. **Archive** learnings for next time.

### Activation phrases
"figure it out", "find a way", "make it work", "just do it", "solve this", "handle it", "I'm stuck", "this isn't working", "too expensive", "need an API key", any detected blocker (auth/rate-limit/dependency/404/etc.).

## Install
**Mac/Linux**
```bash
curl -fsSL https://raw.githubusercontent.com/eugineous/figure-it-out/main/install.sh | bash
```

**Windows PowerShell**
```powershell
irm https://raw.githubusercontent.com/eugineous/figure-it-out/main/install.ps1 | iex
```

**Manual**
1) Download `figure-it-out.md`
2) Copy to your Claude Code skills directory (`~/.claude/skills` or `%USERPROFILE%\.claude\skills`)
3) Restart Claude Code
